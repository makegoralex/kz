# AGGREGATOR 24/7 Landing

Современный адаптивный сайт-визитка для проекта **AGGREGATOR 24/7** (React + Vite + Tailwind CSS + Framer Motion).

## Запуск

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

`npm start` поднимает Node-сервер и раздает содержимое `dist/`.
